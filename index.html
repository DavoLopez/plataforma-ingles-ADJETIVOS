Formación de Adjetivos y Jerarquías B2

En inglés de nivel B2, no existe una regla única para la creación de adjetivos, sino una serie de patrones formados a través de la adición de sufijos y prefijos. Además, cuando empleamos múltiples adjetivos simultáneamente, debemos respetar el estricto orden de posición que dicta la gramática formal.

Para dotar a un sustantivo de un carácter descriptivo, empleamos sufijos específicos:

  • -al, -ial (relacionado con): norm → normal, music → musical.
  • -ic, -ical (naturaleza de / causado por): romance → romantic.
  • -ful (lleno de / que tiene la cualidad): beauty → beautiful, harm → harmful.
  • -less (sin / que carece de): end → endless, speech → speechless.
  • -ish (origen / cualidad atenuada): child → childish, Britain → British.
  • -eous, -ious, -ous (cualidad / naturaleza): adventure → adventurous, ambition → ambitious.
  • -y (estado / aspecto superficial): anger → angry, wealth → wealthy.
  • -like / -ary (semejanza / relación): child → childlike, custom → customary.

La transformación de verbos a adjetivos se asocia a la capacidad de realizar o provocar una acción:

  • -ive (causa de un efecto activo): create → creative, talk → talkative, compete → competitive.
  • -able, -ible (viabilidad / posibilidad): read → readable, break → breakable, rely → reliable.
  • -ful (plenitud): play → playful.
  • -ent, -ant (ejecutor de la acción): obey → obedient, expect → expectant.

Classic vs Classical

Classic: Gran calidad, tradicional, que no pasa de moda.
"A classic blue skirt."

Classical: Referente a la cultura clásica grecorromana o artes formales tradicionales.
"Classical composer Mozart."

Economic vs Economical

Economic: Relativo a las finanzas, industria, política monetaria o comercio.
"Economic policy."

Economical: Ahorrativo, de bajo consumo de combustible o dinero.
"Economical hybrid cars."

Historic vs Historical

Historic: Hito trascendental o sumamente importante en la historia.
"A historic peace agreement."

Historical: Vinculado al estudio formal o documentos del pasado.
"Historical novels."
